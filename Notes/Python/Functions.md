# Functions

## Definition

A function is a reusable block of code that performs a specific task.

## Syntax

```python
def function_name(parameters):
    # code
    return value
```

## Example

```python
def square(x):
    return x * x

print(square(5))
```

## Key Points

- Improves code reusability
- Can accept parameters
- Can return values
- Use `return` to send output back

## Time Complexity

Depends on the operations inside the function.
