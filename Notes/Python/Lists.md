# Lists

## Definition

A list is an ordered, mutable collection of elements.

## Example

```python
numbers = [10,20,30]
```

## Common Methods

- append()
- insert()
- remove()
- pop()
- sort()
- reverse()

## Slicing

```python
numbers[1:4]
```

## Key Points

- Mutable
- Allows duplicate values
- Indexed from 0
