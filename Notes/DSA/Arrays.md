# Arrays

## Definition

An array stores elements of the same data type in contiguous memory.

## Operations

- Traversal
- Searching
- Reverse
- Largest
- Smallest

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Access | O(1) |
| Search | O(n) |
| Traversal | O(n) |
| Reverse | O(n) |
