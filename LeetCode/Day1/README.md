## Problems Solved

### 1480. Running Sum of 1D Array

**Approach**

- Traverse the array.
- Maintain a running sum.
- Store each prefix sum.

**Time Complexity:** O(n)

**Space Complexity:** O(n)

### 1920. Build Array from Permutation

**Approach**

- Create a new array.
- Store `nums[nums[i]]` at each index.
